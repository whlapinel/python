1. Data Types and Variables

x = 3.5
y = int(x)
z = x + y
print(z)

  • B) 7
  • C) 6
  • D) 3.5

for i in range(3, -1, -1):

4. Lists

Question: What is the result of the following code?

list1 = [1, 2, 3, 4]
list2 = list1
list1[0] = 99
print(list2)
  • A) [1, 2, 3, 4]
  • B) [99, 2, 3, 4]
  • C) None
  • D) [1, 2, 3]

  • D) KeyError
